## 🧠 what I actually do

I work in **Venture Capital** — which means I'm supposed to be finding the next great company,
not copy-pasting founder names into a CRM at 11pm.

So I wrote code to do that instead.

Now I spend my time on the part that actually matters:
**talking to interesting people building interesting things.**

The boring stuff? My scripts handle it. They don't complain. They don't need coffee. They're great.

## 🤖 things I've automated so I never have to do them again

| The Soul-Crushing Thing | The Fix |
|---|---|
| Manually sourcing startups from the internet void | [`dealflow`](https://github.com/tashasho/dealflow) — AI finds them, scores them, surfaces the ones worth a call |
| Losing your mind reading SEC Form Ds | [`sec-filings`](https://github.com/tashasho/sec-filings) — maps the US investment landscape so you don't have to |
| Spending 40 mins on Canva to post one tweet | [`social-media-posting-automator`](https://github.com/tashasho/social-media-posting-automator) — end-to-end posting for X & LinkedIn, no pixel-nudging required |
| Gut-feeling on early-stage deep tech risk | [`Nexus9`](https://github.com/tashasho/Nexus9---A-Startup-Risk-Analyser) — let math have an opinion too |
| Staring at Crunchbase wondering what worked | [`Startup-Investments-Patterns`](https://github.com/tashasho/Startup-Investments-Patterns-2013-Crunchbase) — pattern-matched a decade of deals |
| Banging your head on a Claude prompt alone | [`thinkor`](https://github.com/tashasho/thinkor) — think out loud, get unstuck |
| Founders drowning in tools with no system | [`forge`](https://github.com/tashasho/forge) — AI-powered resource orchestration for the ones building |
